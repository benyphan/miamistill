# Miami Gun

Top-down шутер в стиле Hotline Miami, разработанный на Python с использованием библиотеки Arcade.

## Возможности
- Быстрый и хардкорный геймплей
- Процедурная генерация уровней
- Интеллектуальные противники
- Система оружия (пистолет, дробовик)
- Визуальные эффекты (кровь, вспышки, тряска камеры)

## Управление
WASD – движение  
Мышь – прицеливание 
ЛКМ – стрельба
1 / 2 – смена оружия  
P – пауза  
R – рестарт после смерти  

## Запуск
### EXE. Откройте файл: MiamiGun.exe

## Системные требования
- Windows 10/11
- 2 GB RAM
- OpenGL 3.3+
